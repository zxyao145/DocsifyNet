# DocsifyNet
Docsify base on .NET 6.

Build a document site or knowledge base with one click.


# usage

## Windows Service

Download `WindowsService.zip`, and run `#install.bat` to install as Windows Service. 

Run `#uninstall.bat`, DocsifyNetService will delete from windows service.


- 

## Web Service

Download `FrameworkDependency.zip`, and run `dotnet DocsifyNet.dll`.
